Regras de entrada neste arquivo:

Todas as novas entradas devem ser datadas e podem ou não incluir nomes de código para cada versão. Cada entrada deve começar com o caractere "+", seguido da data, entre "[]" e ":". Após, as alterações devem ser descritas da forma mais clara possível.

Alterações realizadas no HBoot:
===============================

+ [11/01/2021]: Anterior a data de 11/01/2021, as alterações não haviam sido documentadas, desde a criação do componente, no final de novembro de 2020.
+ [30/01/2021]: Alteração no código de suporte a FAT16B para não finalizar a execução após não encontrar um arquivo, apenas setar erro e devolver controle para a função solicitante, que vai tratar o erro. O arquivo "Lib/modulos.asm" agora agrupa os módulos built-in e o carregador de módulos, juntamente como estruturas de controle. "Lib/modulos.asm" original teve seu código movido para "Lib/modHBoot.asm", e "Lib/compatDOS.asm" foi renomeado como "Lib/modDOS.asm".
+ [20/02/2021]: Alterações na forma com que o HBoot obtia a memória RAM disponível no boot. O código anterior manipulava até 64 MB e, caso a quantidade de RAM instalada fosse maior, causava um estouro e o código reconhecia como um valor menor de 32 MB (mínimo). O novo código tem limite d detcção de 2^32 bytes.
